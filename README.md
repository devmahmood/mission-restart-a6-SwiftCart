১. null vs undefined
undefined মানে ভেরিয়েবল বানানো হয়েছে কিন্তু তাতে কিছু রাখা হয়নি (অনিশ্চিত)। আর null মানে নিজে থেকেই বলে দেয়া হয়েছে যে ভেরিয়েবলটা এখন "খালি" বা "শূন্য" (নিশ্চিত)।

২. map() vs forEach()
map() মেথড ব্যবহার করে একটা লিস্ট থেকে প্রসেস করে নতুন একটা লিস্ট (অ্যারে) পাওয়া যায়। আর যদি শুধু লুপ চালিয়ে কোনো কাজ করতে চাই (যেমন: কনসোলে কিছু দেখানো), কিন্তু নতুন কোনো লিস্টের দরকার নাই, তাহলে forEach() ব্যবহার করতে হবে।

৩. == vs ===
== শুধু দেখে দুই পাশের মান (Value) এক কি না। কিন্তু === মান এবং ডাটা টাইপ—দুইটাই সমান কি না তা চেক করে। 

৪. async/await-এর গুরুত্ব
আগে ডাটা ফেচ করতে গেলে কোড অনেক প্যাঁচানো লাগত (Promise/Then)।  async/await আসার ফলে কোড পড়া এবং Debug করা অনেক সহজ হয়েছে।

৫. Scope (স্কোপ) কী?
স্কোপ মানে ভেরিয়েবলের সীমানা।

Global: পুরো কোডের যেকোনো জায়গা থেকে পাওয়া যাবে।

Function: শুধু নির্দিষ্ট ফাংশনের ভেতর কাজ করবে।

Block: শুধু নির্দিষ্ট থার্ড ব্র্যাকেট বা লুপের { } ভেতর কাজ করবে।
